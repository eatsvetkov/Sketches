## Modbus RTU Master ESP32 implementation

`main.py` example for implementation Modbus RTU protocol on esp32 with using uModBus library that is based on pycom-modbus from pycom.

### User files:

- boot.py
- main.py

### Library files:

- uModBusConst.py
- uModBusFunctions.py
- uModBusSerial.py
- uModBusTCP.py
